# 日文文法指南

原作者 Tae Kim：

我寫這個指南是爲了能夠幫大家以一種直觀的方法學習日文，從日文而非英文的角度出發來理解日文文法。

本指南使用創作共用許可發佈，您可以向非商業使用者自由的複製和散佈這份指南。
